# 获取帮助

请根据问题类型选择合适的渠道：

- 配置使用、方案讨论和一般问题：使用 [GitHub Discussions](https://github.com/MiChongs/WutherCore/discussions)。
- 可以稳定复现的缺陷：使用 [Bug Report](https://github.com/MiChongs/WutherCore/issues/new?template=bug_report.yml)。
- 功能建议：使用 [Feature Request](https://github.com/MiChongs/WutherCore/issues/new?template=feature_request.yml)。
- 安全漏洞：按照 [SECURITY.md](SECURITY.md) 私下报告。

提问前请先搜索现有 Issue 和 Discussion。公开内容中应删除订阅地址、节点凭据、私钥、访问令牌和可识别个人网络环境的信息。

这是社区维护的开源项目，不提供响应时间保证。信息完整、能够复现、范围明确的问题会更容易得到处理。
