## 文档约定

- 命令默认从仓库根目录执行。
- 配置片段使用当前 `version: 1` 格式。
- “已实现”表示仓库中存在可执行代码路径和测试，不表示与所有第三方服务端组合均已验证。
- 平台相关功能必须同时说明操作系统、权限、流量接管方式和回滚路径。
- 示例中的域名、订阅地址、密码、UUID 和密钥都是占位符，不应直接用于生产环境。
